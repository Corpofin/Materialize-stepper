INSTALLING
Direct download
  • CSS:
    • mstepper.min.css minified, or
    • mstepper.css un-minified
  • JavaScript:
    • mstepper.min.js minified, or
    • mstepper.js un-minified
NPM
npm install --save materialize-stepper@beta
YARN
yarn add materialize-stepper@beta
CDN

<!-- CSS -->
<link rel="stylesheet" href="https://unpkg.com/materialize-stepper@{version}/dist/mstepper.min.css">
      

<!-- JS -->
<script src="https://unpkg.com/materialize-stepper@{version}/dist/mstepper.min.js">
